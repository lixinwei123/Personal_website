Android Engineer Intern

The purpose of the application was to provide an efficient and user friendly way to track the total amount of data a mobile has used based on the country of the device. All data gathered are stored locally with Users consent.

Some Features include:
1.A pie chart graph that shows the total amount of data used and the amount of data left.
2.User can view history of the data usage since the past 3 months
3.The User can change the country they are located in any time
4.The user can have quick access to IT-ticket support in case their data usage is not enough or there are issues.
